## Run Locally

Clone the project

```bash
git clone https://github.com/Thamim1626/new_booking_app.git
```

Go to the project directory

```bash
  cd backend/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

### Backend

`mongo` - database

`port` - running port

`RAZORPAY_KEY_ID`

`RAZORPAY_KEY_SECRET`

`CURRENT_ENVIRONMENT`

`SMTP_USERNAME`

`SMTP_PASSWORD`

`SMTP_PORT`

### Frontend

`RAZORPAY_KEY_ID`

`RAZORPAY_KEY_SECRET`

`PORT`

## Features

- Login and Sign up using Mobile OTP
- Search engine to search Flights
- Sorting of flights as his price
- Apply the coupon on your cart
- Purchase the Product
- Do payment with any type of method(Card/UPI/Net-banking)
